Transition

it is used to change the properties of an element which helps to animate the element 
                syntax
                {
                    transition-property: value;
                    transition-duration: value;
                    transition-timing-function: value;
                    transition-delay: value;
                }

Note : Transition will apply when elements are hovered

this is a div

Types of transition-timing-function

  • ease
  • ease-in
  • ease-out
  • ease-in-out
  • linear
  • cubic-bezier
  • steps
ease: initialy slow then fast
ease-in: initialy slow then fast
ease-out: initialy fast then slow
ease-in-out: initialy slow then fast then slow
linear: constant speed
cubic-bezier: custom
steps: number of steps
