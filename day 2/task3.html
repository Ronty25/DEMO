list

it is used to display list of items
  1. Order list
  2. Unordered list
  3. Data definition/Description list
  4. Nested list

Ordered list

It is used to display a list in a specific order

ol tag is used to create an ordered list

The default number is (1,2,3...)

Type attribute helps to change the default order(type= "A", "a", "I", "i", "1")

Reverse attribute helps to reverse the order

start attribute helps to start the list from a specific number

Unordered list

it is used to display a list without any order

ul is used to create an unoredered list

style attribute helps to change the default order (list-style "disc", "square", "circle")

type attribute can also be used to change the default order(type="disc","circle","square")

Data definition/Description list

it is used to display list of terms and their description

dl tag is used to create data definition list

dt tag is used to define term

dd tag is used to define description
