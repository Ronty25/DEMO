Css positions:

it is used to change the position of an element 
            synntax:
            {
               position: value; 
            }

There are 5 types of positions

  • static - default
  • relative
  • absolute
  • fixed
  • sticky

Static position

it is used to keep the element in it's original positions

static position is default

top, right, bottom, left cant be used

Relative Position

it is used to change the position of an element from it's original position

top, right, bottom, left can be used

Fixed Position

It is used to keep the element in a fixed position

top, right, bottom, left cant be used

Css Positions

Sticky Position

It is used to keep element in a fixed position when scrolling

top, right, bottom, left cant be used

it is a combination of relative and fixed

Absolute Position

It is used to keep the element in a parent element

top, right, bottom, left can be used

Note: it is not a child element, then it will be positionrd relative to the body

parent element should be in relativ e position
